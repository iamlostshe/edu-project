# edu-project

## How to launch

1. Clone repository:

```
git clone https://github.com/iamlostshe/edu-project.git
cd edu-project
```

2. Create and activate `venv`:

```
python3 -m venv venv
. venv/bin/activate
```

> [!NOTE]
> Second comand for windows systems is:
> ```
> venv\Scripts\activate
> ```

3. Install requirements:

```
pip install -r requirements.txt
```

4. Launch app:

```
python3 main.py
```

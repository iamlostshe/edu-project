Ниже будут подробно описаны стандарты по оформлению и загрузке конфигурационного файла для создания курса на платформе `edu-project`.

### Структура

Конфиг пишется в `json` файле, для него вы можете использовать любое название, рекомендуется `index.json` (в ином случае - ошибка оформления NAM001).

Таким образом структура проекта может быть такой:

```bash
.
├── LICENSE    - Файл лицензии
└── index.json - Основной файл
```

> [!INFO]
>
> Также в вашем репозитории может быть файл лицензии, вот полезная ссылка по теме:
>
> [Как выбрать лицензию?](https://doka.guide/recipes/github-choose-license/)

### Название курса - `name`

Название должно быть сторокой, может содержать любые символы:

```json
{
    "name": "Ультимативный курс по Python",
    ...
}
```

### Версия - `version`

Версия - это необязательный парметр, строка, которая может содержать любые символы. Согласно VER001 версия может быть указана в 2-х форматах:

- Дата: `19.03.2025` или `2025.03.19`.
- Обычная версия: `0.3.16` или `2.123`.

```json
{
    ...
    "version": "0.0.1",
    ...
}
```

> [!INFO]
>
> Версию не обязательно менять при каждом изменении (коммите) на GitHub, как только что-то в репозитории будет изменено - это сразу отобразится на сайте.

### Авторы - `authors`

Авторы - обязательный параметр, список со словарями (см. пример ниже):

```json
{
    ...
    "authors": [
        {"name": "Melcikov Ivan", "email": "vanamelcikov7275@gmail.com"},
        {"name": "Example Author", "email": "example@example.com"}
    ],
    ...
}
```

> [!INFO]
>
> Поле `email` является не обязательным.

### Описание - `description`

Описание - строка, краткое описание проекта:

```json
{
    ...
    "description": "Курс по Python от олимпиадников."
    ...
}
```

### Теги - `keywords`

Теги - грубо говоря, слова, при вводе которых в поиск выйдет ваш курс.

```json
{
    ...
    "keywords": [
        "python", "питон",
        "ЯП", "Языки программирования",
        "программирование", "информатика"
    ],
    ...
}
```

### Предмет - `subject`

```json
{
    ...
    "subject": "Информатика",
    ...
}
```

> [!INFO]
>
> Нужен для аналитики/сортировки.

### Итоговый файл

```json
{
    "name": "Ультимативный курс по Python",
    "version": "0.0.1",
    "authors": [
        {"name": "Melcikov Ivan", "email": "vanamelcikov7275@gmail.com"},
        {"name": "Example Author", "email": "example@example.com"}
    ],
    "description": "Курс по Python от олимпиадников.",
    "keywords": [
        "python", "питон",
        "ЯП", "Языки программирования",
        "программирование", "информатика"
    ],
    "subject": "Информатика"
}
```

### Загрузка курса в систему

1. Создайте новый публичный репозиторий на [github.com](https://github.com), загрузите в него ваш конфигурационный файл и лиценцию, если такая есть (см. [структура](#структура)).

2. Написать [админу в Telegram](https://t.me/iamlostshe), прекрепить ссылку на созданный ранее репозиторий.

3. После модерации ваш курс появится в `edu-project`.
